# Danya | Tatoo — сайт-портфолио

Готовый адаптивный одностраничный сайт для тату-мастера: тёмная эстетика, анимации, мобильное меню, каталог работ с фильтрами, лайтбокс, QR-коды для Telegram / VK / почты / адреса.

## Что внутри

- `index.html` — структура сайта
- `styles.css` — стили, адаптив и анимации
- `content.js` — весь текстовый контент, карточки и контакты
- `app.js` — рендер секций, фильтры, лайтбокс, мобильное меню
- `assets/img` — оптимизированные изображения работ и логотип
- `assets/qr` — QR-коды для перехода с телефона

## Как открыть

Просто откройте `index.html` в браузере.
